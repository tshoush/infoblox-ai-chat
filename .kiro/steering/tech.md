# Technology Stack

## Backend

- **Framework**: Flask 3.0.3 (Python web framework)
- **HTTP Client**: requests 2.32.3 (for WAPI communication)
- **Data Formatting**: tabulate 0.9.0 (for structured output)
- **Language**: Python 3.12
- **Authentication**: Basic Auth for Infoblox WAPI
- **API Protocol**: REST over HTTPS

## Frontend

- **Framework**: React 18.3.1
- **UI Components**: react-autosuggest 10.1.0 (for intelligent input suggestions)
- **Build System**: Standard React build tools
- **Styling**: CSS with Marriott-inspired design system

## External Integrations

### Infoblox WAPI
- **Version**: v2.13.1 (configurable)
- **Protocol**: HTTPS REST API
- **Authentication**: Basic authentication with admin credentials
- **SSL Verification**: Disabled by default (configurable)

### LLM Providers
- **Supported**: OpenAI, Claude, Grok, Llama, Gemini
- **Authentication**: API key-based
- **Custom Endpoints**: Configurable base URLs for local/custom models

## Development Tools

- **Testing**: unittest (Python), Jest + React Testing Library (Frontend)
- **Containerization**: Docker with Python 3.12 base image
- **Version Control**: Git
- **Documentation**: RAG system with PDF, YAML, and HTML sources

## Common Commands

### Setup and Installation
```bash
# Run initial setup (interactive)
./setup.sh

# Install backend dependencies
pip install -r backend/requirements.txt

# Install frontend dependencies
cd frontend && npm install
```

### Development
```bash
# Start backend server
python backend/app.py

# Start frontend development server
cd frontend && npm start

# Run backend tests
python -m unittest discover backend/tests

# Run frontend tests
cd frontend && npm test
```

### Docker Operations
```bash
# Build container
docker build -t iaci .

# Run container
docker run -p 5000:5000 iaci
```

### Configuration
- **Backend Config**: `backend/config.json` (generated by setup.sh)
- **Vocabulary**: `vocabulary.json` (updated during runtime)
- **Documentation**: `rag_docs/` directory (populated by setup.sh)
